Data Management

Export creates a JSON backup. Import restores from a backup file. Reset clears all custom data and reloads defaults.

How it works

  • • Changes are saved to your browser's localStorage
  • • The portfolio reads from localStorage automatically — changes appear instantly when you refresh
  • • Use Export to download a backup JSON before clearing browser data
  • • Use Import on another device to sync your data
  • • The admin link is in the portfolio footer (small "Admin" link)
